## LINUX Commands 
** Why we need Linux commands ?**
- Because Linux Mostly controlled through a terminal(Command lines) Insted of clicking Buttons 

## Commands
** File And Directory **
- ls : List of files
- cd : change Directory 
- pwd : shows the current path
- mkdir : Create folder
- rm : delete file
- cp  a b: copy file
- mv  a b: move/rename the file
- cat : view file content
