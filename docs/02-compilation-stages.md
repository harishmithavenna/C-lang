## From the abouve Compilation Stages 
** PreProcessor Stage **
- It Removes the comment Lines 
- It Expands macros
- Include header files 
- Handles Conditional Compilation 

** Tanslator stage/Compiler **
- it checks the Syntax of the code
- converts into assembly code

** Assembler Stage**
- It converts Assembly code into Machine Code

** Linker Stage**
- it links Obj Files with Libraries

**a.out(Executable Stage)**
- Final Runnable file 
- contains Complete Machine code
