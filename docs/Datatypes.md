#Data Types Of C 
## What is Data Types?
**The Data type defines the Property of Variable** as to Know What Kind of Data will the variable stores 
#why we Use Data types?
- Memory Management  ** It tells that Compiler How much data it will allocate 
Example - 
- int =4 Bytes of data
- char = 1 Byte of data etc
- Defines What operations are valid 
 that means you can do any operation on numbers (+,-,*) but not directly on characters
- Improves code readability 

## types Of data type
Classified into ----------
- Basic Data Types -- it includes int ,float,char,double
- Derived Data types -- Pointers,arrays ,functions
- User defined Data types-
- Enm,Union,struct
