# Arrays
- What is an Array?
- Array is Collection of Homogenous data types in a ** Contiguous memory location ** 
- ** Each element is accessed by an index **
- Works very efficiently with loops
## Where We Use this Arrays?
-Student Management System
-Employee Salary Database
-Temperature Recording
-Banking
## Why Is the Arrays
-Store Large Amounts of Similar Data
-Easy Processing Using Loops
-Easy Searching

## Syntax of the Array 
- data_type array_name[size];
- example
Int a[5]={1,2,3,4,5};
- Negative index is also possible in arrays

## Array Initilization - 3 ways
- Proper Initilization-  Int a[5]={1,2,3,4,5};
- Excess Intilization-   Int a[5]={1,2,3,4,5,6,7};
- Partial Intilization-  Int a[5]={1,2,3};

 ## Internal Working of an Array
- array name represent base address 
- array is internally treated as ** constant pointer **
- a[i] in array= *(a+i) in pointer
